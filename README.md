# გზა · Gza — Tour Guide to Georgia

Bilingual (ქართული / English) tour guide for Georgia.
Curated routes, useful tips and a trip planner. Built with React + Vite.

## Run

```bash
npm install
npm run dev
```

Then open the printed local URL.

## Files

- `index.html` — entry HTML
- `src/main.jsx` — React entry point
- `src/App.jsx` — entire app (nav, pages, planner, i18n)
- `src/index.css` — design tokens + base styles
- `src/App.css` — component styles
- `src/hero-georgia.jpg` — hero image
